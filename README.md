# О проекте
## Описание проекта 
Разработка веб-приложения для генерации персонажей ролевых игр с использованием ИИ
## Цели и задачи 
Создание системы генерации персонажей, аутентификации пользователей, экспорта данных
## Технологии 
Python, Flask, SQLAlchemy, AI API (OpenAI), HTML/CSS
## Функциональные возможности
- Регистрация и авторизация пользователей
- Генерация персонажей по текстовому описанию
- Редактирование характеристик персонажей
- Генерация изображений персонажей
- Экспорт данных в TXT/PDF форматах
## План реализации
- Лабораторная работа 1: Описание проекта и план работ
- Лабораторная работа 2: Разработка архитектуры приложения и аутентификации
- Лабораторная работа 3: Реализация генерации персонажей и бизнес-логики
- Лабораторная работа 4: Реализация функций редактирования и работы с изображениями
- Лабораторная работа 5: Реализация экспорта данных и финальная доработка
- Лабораторная работа 6: Инструкция по запуску и демонстрация работы

### Настройка

- Переходим в папку проекта `cd character_generator`
- Создаем виртуальное окружение `python -m venv venv`
- Активируем его `venv\Scripts\activate`
- Устанавливаем зависимости `pip install -r requirements.txt`
